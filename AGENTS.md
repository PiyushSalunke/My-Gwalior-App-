# Custom Instructions for My Gwalior / Civic Pulse App

## Mobile-First UI/UX Design Priority
- **High Priority Requirement**: Every feature, layout, view, and interactive element MUST be designed and optimized with **mobile view as a first-class citizen**.
- When designing UI/UX, always ensure perfect readability, fluid tap targets (minimum 44px), compact padding, elegant grid adaptation, and seamless navigation on mobile devices.
- Always offer customizable columns for grids (e.g., toggle between 1 or 2 parallel cards in mobile view) so users can customize density according to their screen sizes.
